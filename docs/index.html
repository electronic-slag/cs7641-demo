System Features

🎯

Cold-Start Handling

Attribute-based clustering allows new songs to be recommendable immediately, even without interaction data.

🧠

Lightweight & Interpretable Model

Uses LightFM with sparse cluster and audio-emotion features, avoiding deep models to preserve interpretability.

🔬

Multi-Modal Feature Integration

Combines emotional, audio, and metadata to enrich song representation and improve recommendation quality.

🧪

Clustering-Assisted Recall

Clusters group similar songs by mood and sound, supporting more diverse and semantically meaningful recommendations.

📈

Transparent Evaluation

Achieved a 45% lift in Precision@10 over baseline; diagnostic plots provided to interpret model decisions.

🧰

Simple Web Interface

Deployed via Streamlit with basic song search, cluster-based suggestions, and visualization of user-song matches.
